Bonzo Assistant Sample Deploy Note

This is a bundled sample document for the first RAG demo.

Key points:

- For a beginner-friendly hosted setup, Azure App Service is a simple first choice.
- If you want to avoid paid API usage while learning, keep `MOCK_MODE=true`.
- When the app is containerized, Streamlit should listen on `0.0.0.0` and port `8501`.
- A custom startup command can be useful in hosted environments that need an explicit run command.
